# SOLIDBankApp

## Definition

Simple application for Bank

## Run

 - Download jar file.
 - Open terminal.
 - Run it.

```
java -jar SOLIDBankApp-0.0.1-SNAPSHOT.jar
```

## Functionality
Right now this application can do 4 from 7 actions
 - Show all accounts of client.
 - Create new account for client.
 - Help message.
 - And close of application
